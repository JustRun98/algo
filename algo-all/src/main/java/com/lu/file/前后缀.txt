如果前缀数组中相同的值出现了两次，表示数组两端中间和为0
相同的，如果差值为一个数，表示数组两端中间和为那个数
------------------------------------------------------------------------
前缀和初始化模板：
this.prefix = new int[nums.size() + 1];
for (int i = 0; i < nums.size(); i++) {
    prefix[i + 1] = prefix[i] + nums.get(i);
}

这样对于数组:
求下标 0-8的和：
prefix[8+1] - prefix[0]
求下标 60 - 100 的和:
prefix[101] - prefix[60]
